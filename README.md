# 專案摘要說明

  專案目標:
  
   * 使用 JSPM 管理 Client 端所需之 JavaScript Module
   * 使用 JSPM 管理 Client 端之 JavaScript 程式「打包」成單一檔案

## 安裝作業

    ```
    $ npm install
    
    $ jspm install
    ```

## 執行作業

### 啟動 Client

    ```
    $ npm run serve
    ```

### 啟動 Server

    ```
    $ npm start
    ```
